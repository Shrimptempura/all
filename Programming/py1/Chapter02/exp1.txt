print(Hello Python)
->  따음표 생략시 오류 발생

print('오늘 걸음 수', 8000, '걸음')
-> 문자열과 쉼표로 구분된 숫자나 문자열은 print() 함수에서 같은 줄에 함께 출력됨

print('Hello ' * 2)
-> Hello Hello
-> *(곱셈) 연산자는 출력횟수를 제어가 가능하다.

변수 : 컴퓨터가 메모리에 값을 저장해 높은 위치의 이름
식별자 : 사용자가 정의하는 변수나 함수에 대해 서로 구별되는 이름

메모리 공간 : 보통 16진수의 주소

<변수의 생성과 식별자>

name = '홍길동'
name : 변수 이름
= : 할당 연산자
'홍길동' : 문자열

'홍길동'이나 27과 같은 데이터 값 : 리터널(literal)
