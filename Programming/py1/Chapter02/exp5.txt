<비트 연산자>
& : 비트 단위 AND : 모두 1이면 1, 아니면 0
| : 비트 단위 OR : 하나라도 1이면 1, 아니면 0
^ : 비트 단위 XOR : 두개 모두 같으면 0, 아니면 1
~ : 비트 단위 NOT : 0은 1로 만들고, 1은 0으로 만든다.
<< : 비트 단위 왼쪽으로 이동 : 지정 개수만큼 모든 비트 이동
>> : 비트 단위 오른쪽으로 이동 : 지정 개수만큼 모든 비트 이동

bin(9)      # 2진수 형식 출력
