<자동 타입 변환(promotion)>
구현 객체가 인터페이스 타입으로 변환되는 것은 자동 타입 변환(promotion)에 해당된다.
자동 타입 변환은 프로그램 실행 도중에 자동적으로 타입 변환이 일어나는 것을 말한다.
                자동 타입 변환
    인터페이스 변수 = 구현 객체;

인터페이스 구현 클래스를 상속해서 자식 클래스를 만들었다면 자식 객체 역시 인터페이스 타입으로
자동 타입 변환시킬 수 있다.

D : 자식클래스 -> B : 구현 클래스 -> A : 인터페이스
E : 자식클래스 -> C : 구현 클래스 -> A : 인터페이스

B b = new B();              A a1 = b; (가능)
C c = new C();      -->     A a2 = c; (가능)
D d = new D();              A a3 = d; (가능)
E e = enw E();              A a4 = e; (가능)

자동 타입 변환을 이용하면 필드의 다형성과 매개 변수의 다형성을 구현할 수 있다.
필드와 매개 변수의 타입을 인터페이스로 선언하면 여기에 다양한 구현 객체를 대입해서 
실행 결과를 다양하게 만들수 있다.
