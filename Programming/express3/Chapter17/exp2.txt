<calloc()>
0으로 초기화된 동적 메모리를 할당한다. 따라서 초기화된 메모리 블록을 얻을 때는 
calloc()을 사용하는 것이 편리하다. 또한 바이트 단위가 아닌 항목 단위로 메모리를 할당한다.

int *p\  
p = (int *)calloc(5, sizeof(int));      5 -> 항목의 개수

<realloc()>
할당하였던 메모리 블록의 크기를 변경한다.

int *p;
p = (int *)malloc(5 * sizeof(int));
p = realloc(p, 7 * sizeof(int));
        p = 기존의 동적 메모리,
        7 * sizeof(int) = 새로운 크기

    첫 번째 매개 변수는 기존에 동적 할당된 메모리 블록을 가리키는 포인터이다.
    
    두번째 매개변수는 새로운 메모리 블록의 크기(바이트 단위)이다.
    기존의 데이터 값은 유지된다.
