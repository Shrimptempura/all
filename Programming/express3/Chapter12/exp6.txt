// 문자열 여러개 저장 -> 2차원 문자 배열을 사용하는 방법
char s[3][6] = {"init", "open", "close"};
    --> printf("%s", s[0]);     // "init"가 출력된다.

// 문자 포인터 배열 사용법
char *s[3] = {"init", "open", "close"};
s는 3개의 포인터를 요소를 가지는 배열이다. 각 포인터는 메모리에 저장된 문자열 상수의
주소를 가지고 있다. 문자열 상수는 문자열의 크기만큼만 자리를 차지한다.
    --> 따라서 공간을 절약할 수 있다.
    단점) 한번 문자열이 결정되면 문자열의 내용을 변경할 수 없다.
