gets_s(버퍼, 버퍼크기);
    char *gets_s(char *_Buffer, rsize_t _Size);
    성공하면 입력된 문자열을 반환, 실패하면 NULL을 반환

puts(문자열)
int puts(char const *_Buffer);
성공하면 음수가 아닌 값을 반환, 실패하면 EOF(-1)를 반환

ex)
char buf[100];
gets_s(buf, 100);       // 사용자로부터 한줄을 입력받는다, 최대 입력 개수는 100이다.
puts(buf);              // 한줄 출력

gets_s() : 개행이 나올때까지 한 줄의 라인을 문자열로 입력받음.
            문자열에 '개행'은 추가되지 않음. 대신 자동으로 '널'추가
            성공하면 입력된 문자열을 반환, 실패하면 NULL을 반환


puts() : 출력 함수, 문자열 끝에 있는 '널'은 - > '개행'으로 변경됨
        성공하면 음수가 아닌 값을 반환, 실패하면 EOF(-1)를 반환
